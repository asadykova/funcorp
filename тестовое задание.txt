1/ получу список полномочий пользователя выполнив GET запрос https://example.com/calc/permissions?userId={{userId}}

{
    "data": {
		"permissions": [

2/ выполнив аутентификацию под пользователем c {{userId}}, проверю выполнение следующих POST запросов:
 https://example.com/calc/addition
 https://example.com/calc/subtraction
 https://example.com/calc/multiplication
 https://example.com/calc/division
 
 тело запроса будет выглядить примерно следующим образом:
{
	"a": {{a}},
	"b": {{b}}
}

 в каждом запросе имеются тесты для сравнения полученного результата с ожидаемым результатом:
 - получу результат из тела ответа: 
									let result = pm.response.json().result;
 - получу значения переменных окружения, которые передавались в тело запроса:
									let value1 = pm.environment.get("a");
									let value2 = pm.environment.get("b");
 - сравню ожидаемый результат с полученным результатом из тела ответа (приведен пример проверки результата умножения):
									pm.expect(result).to.eql(value1*value2);
							
3/ учитывая полученный список полномочий из п.1, запросы /calc/addition, /calc/subtraction и /calc/division должны выполниться успешно и вернуть резальтат, полученный резальтат сравню с ожидаемым
4/ ожидаю, что запрос /calc/multiplication вернет ошибку с примерным текстом "user has not permission ..", и тест сравнения полученного резальтата с ожидаемым результатом упадет
5/ предположим, что тест в запросе /calc/multiplication выполнился успешно, хотя не должен был. 
Баг-репорт будет составлен следующим образом:
